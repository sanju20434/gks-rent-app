# User Stories and Use Cases

## Actors

- Admin/Store Operator
- Client

## User Stories

1. As an admin, I want to create and manage clients so I can track rental activity by customer.
2. As an admin, I want to manage item stock and rental rates so billing is accurate.
3. As an admin, I want to create project records for clients so bookings and transactions are project-linked.
4. As an admin, I want to issue material OUT and material IN records so movement is auditable.
5. As an admin, I want to generate bills and record payments so dues can be tracked.
6. As an admin, I want to view a client ledger so I can verify debit/credit history.
7. As an admin, I want to send WhatsApp due reminders so collection is faster.
8. As an admin, I want to generate and send QR-based payment links so clients can pay quickly.
9. As an admin, I want a dashboard summary so I can monitor outstanding and recent activity.

## Primary Use Cases

- UC-01: Add/Edit/Remove client
- UC-02: Add/Edit/Remove items
- UC-03: Create project against client
- UC-04: Create booking
- UC-05: Material OUT transaction
- UC-06: Material IN + invoice generation
- UC-07: Record payment/deposit
- UC-08: Send WhatsApp custom or total due message
- UC-09: Generate and share QR code
- UC-10: View dashboard and ledger reports
